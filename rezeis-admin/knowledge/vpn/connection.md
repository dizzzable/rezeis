# Настройка подключения

## Android (HAPP)

1. Установи приложение HAPP из раздела «Приложения».
2. Открой приложение и нажми «Авторизоваться».
3. Подтверди через Telegram или отсканируй QR-код из личного кабинета.
4. После авторизации подключение установится автоматически.
5. Готово! Ты в сети.

## iOS (Incy)

1. Установи Incy по инструкции из личного кабинета.
2. Открой приложение и авторизуйся.
3. Нажми кнопку подключения.
4. Первое подключение может запросить разрешение — разреши.

## macOS (FlClashX)

1. Установи FlClashX из раздела «Приложения».
2. Открой приложение.
3. Авторизуйся через Telegram или QR-код.
4. Нажми «Подключиться» в строке меню.

## Windows

1. Скачай приложение для Windows из раздела «Приложения».
2. Установи и открой.
3. Авторизуйся привычным способом.
4. Нажми «Подключиться».

## Скорость и производительность

- Для максимальной скорости используй проводное подключение (Ethernet).
- На мобильных сетях скорость зависит от качества сигнала.
- Приложение автоматически выбирает оптимальный сервер.
