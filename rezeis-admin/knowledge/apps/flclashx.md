# FlClashX (Rezeis)

**FlClashX** — это официальное приложение Rezeis для macOS. Лёгкое, удобное и надёжное.

## Как установить

1. Скачай FlClashX из раздела **Приложения** в боте или на сайте.
2. Перемести приложение в папку **Программы** (Applications).
3. Открой FlClashX — при первом запуске система может попросить подтверждение. Нажми «Открыть».
4. Авторизуйся через Telegram или по QR-коду.

## Основные возможности

- Родной интерфейс для macOS
- Поддержка всех актуальных версий macOS
- Автоматический выбор сервера
- Статус-бар меню для быстрого доступа
- Запуск при старте системы

## Частые вопросы

**macOS говорит, что приложение из непроверенного разработчика.**
Это нормально. Нажми «Открыть» в диалоге — приложение безопасно и проверено.

**Как добавить FlClashX в автозагрузку?**
Открой настройки приложения и включи «Запускать при старте системы».
